# IFSOL - Incubadora de Economia Solidária

## Portal unificado de informações sobre a IFSOL!
<img width="1902" height="828" alt="image" src="https://github.com/user-attachments/assets/18a83602-c0ca-42ae-b82b-af174cf5d9d4" />

A IFSOL é uma incubadora tecnológica com foco na economia solidária e que é distribuída por vários polos do IFRN. Para isso foi desenvolvido o site da IFSOL, com o intuito de servir como portal unificado de informações sobre as atividades de extensão e pesquisa realizadas pela incubadora. 
No site será possível verificar as novidades sobre a IFSOL, ter acesso a novos editais e artigos científicos postados, galeria de fotos de eventos e muito mais!

Acesse o site a partir desse link: https://pedrolpmedeiros.github.io/ifSOL/
